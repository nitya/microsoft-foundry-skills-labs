# Workshop: Build & Optimize AI Agents On Microsoft Foundry With Azure Skills

This is a hands-on workshop to explore and use [Azure Skills](https://learn.microsoft.com/en-us/azure/developer/azure-skills/) with the [GitHub Copilot](https://docs.github.com/en/copilot) AI coding assistant on [VS Code](https://code.visualstudio.com/docs/copilot/overview) - to create, build, deploy, observe, and troubleshoot an AI agent on [Microosft Foundry](https://learn.microsoft.com/en-us/azure/foundry/what-is-foundry).

## 1. Pre-Requisites

1. A GitHub account - with GitHub Codespaces quota
1. A GitHub Copilot subscription - ideally Pro (Claude models)
1. An Azure subscription - with Foundry Models quota
1. Familiarity with VS Code, Git, Python & CLI usage


## 2. Learning Objectives

By the end of this workshop you should be able to:

1. Use Copilot CLI to generate and manage Azure rsources
1. Apply`microsoft-foundry` skill to buid-deploy-observe agents
1. Create a retail AI chatbot with custom data & tools
1. Deploy it as a hosted agent to Microsoft Foundry
